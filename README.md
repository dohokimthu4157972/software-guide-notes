## Windows Software Download Safety Checklist

Before installing software on Windows, it is useful to check several basic details:

1. Confirm that the download page looks like an official or trusted source.
2. Check whether the page uses HTTPS.
3. Avoid installers that force unrelated bundled software.
4. Review the file name before opening the installer.
5. Keep the software updated through a reliable update channel.
